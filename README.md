# Квиз-студия MVP

Веб-приложение для создания и проведения live-квизов. Организатор собирает вопросы, запускает комнату и управляет ходом игры, а участники подключаются по коду и отвечают на текущий вопрос в реальном времени.

## Стек

- React + Vite
- Node.js + Express
- Socket.IO
- JSON-хранилище для MVP-данных

## Быстрый запуск

```bash
npm install
npm run seed
npm run dev
```

Клиент: `http://127.0.0.1:5173`

Сервер: `http://127.0.0.1:4000`

Демо-организатор: `org@quiz.local` / `password123`

Демо-участник: `player@quiz.local` / `password123`

## Переменные окружения

Скопируйте `.env.example` в `.env` и задайте свои значения:

```bash
PORT=4000
JWT_SECRET=replace-with-a-long-random-string
CLIENT_ORIGIN=http://127.0.0.1:5173
```

## Возможности MVP

- регистрация и авторизация организаторов и участников;
- создание, редактирование и запуск квизов;
- вопросы с одиночным или множественным выбором ответа;
- подключение участников к комнате по коду;
- синхронизация состояния комнаты через Socket.IO;
- автоматический переход к следующему вопросу после ответов участников;
- итоговый лидерборд после завершения квиза;
- история проведенных и пройденных квизов.

## Команды

```bash
npm run dev      # сервер и клиент в режиме разработки
npm run build    # сборка фронтенда
npm run start    # запуск API-сервера
npm run lint     # проверка кода
npm run seed     # сброс демо-данных
```
